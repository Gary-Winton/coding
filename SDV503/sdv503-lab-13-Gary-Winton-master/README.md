## :page_facing_up: Lab 5 Description

:arrow_right: `Create a function that groups an array of numbers based on a length parameter. The length represents the max length of each sub-array.`

Example: 
```javascript
    const result = MyFunction([1, 2, 3, 4], 2)
    console.log(result) //[[1, 3], [2, 4]]
``` 
```javascript
   const result = MyFunction([1, 2, 3, 4, 5, 6, 7], 4)
   console.log(result) //[[1, 3, 5, 7], [2, 4, 6]]
```
```javascript
   const result = MyFunction([1, 2, 3, 4, 5], 1)
   console.log(result) //[[1], [2], [3], [4], [5]]
```
```javascript
   const result = MyFunction([1, 2, 3, 4, 5, 6], 4)
   console.log(result) //[[1, 3, 5], [2, 4, 6]]
```
:arrow_right: `Create a function to reverse an 2D array. Replace the 0s with 1s and vice versa.`

Example:
```javascript
   const result = MyFunction([
  [1, 0, 0],
  [0, 1, 0],
  [0, 0, 1]
])
   console.log(result) // [
  [0, 1, 1],
  [1, 0, 1],
  [1, 1, 0]
]
```
