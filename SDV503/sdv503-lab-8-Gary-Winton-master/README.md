## :page_facing_up: Lab  Description

:arrow_right: `Write a function that returns true if an object is empty, and false otherwise.`

Example: 
```javascript
    const result = MyFunction({})
    console.log(result) // true
``` 
```javascript
   const result = MyFunction({ a: 1 })
   console.log(result) // false
```

:arrow_right: `Write a function that converts an object into an array, where each element represents a key-value pair.
Return an empty array if the object is empty.`

Example:
```javascript
   const result = MyFunction({ a: 1, b: 2 })
   console.log(result) // [["a", 1], ["b", 2]]
```
```javascript
   const result = MyFunction({ cars: 15, drivers: 12 })
   console.log(result) // [["cars", 15], ["drivers", 12]]
```
```javascript
   const result = MyFunction({})
   console.log(result) // []
```
:arrow_right: `Create a function that takes an object and returns the keys and values as separate arrays.`

Example:
```javascript
   const result = MyFunction({ a: 1, b: 2, c: 3 })
   console.log(result) // [["a", "b", "c"], [1, 2, 3]]
```
```javascript
   const result = MyFunction({ a: "NMIT", b: "RIT", c: "CSU" })
   console.log(result) // [["a", "b", "c"], ["NMIT", "RIT", "CSU"]]
```
```javascript
   const result = MyFunction({ key1: true, key2: false, key3: undefined })
   console.log(result) // [["key1", "key2", "key3"], [true, false, undefined]]
```
