## :page_facing_up: Lab 9 Description

:arrow_right: `Create a function that takes three arguments (subarr, numitem, item) and returns an array containing subarr subarrays (e.g. [[], [], []]), each containing numitem number of item.`

* `subarr: Number of subarrays contained within the parent array.`
* `numitem: Number of items contained within each subarray.`
* `item: Item contained with each subarray.`

Example: 
```javascript
    const result = MyFunction(3, 2, 1.7) 
    console.log(result) //[[1.7, 1.7], [1.7,1.7], [1.7, 1.7]] 
``` 
```javascript
   const result = MyFunction(4,1,'SDV503')
   console.log(result) //[['SDV503'], ['SDV503'], [SDV503], [SDV503]]
```
```javascript
   const result = MyFunction(1,3,'Hello World!')
   console.log(result) //[['Hello World!', 'Hello World!', 'Hello World!']]
```

:arrow_right: `Create a function that takes a string and calculates the number of letters and digits within it. Return the result as an object. (spaces and symbols are not inculded)`

Example:
```javascript
   const result = MyFunction("Hello World!")
   console.log(result) //{letters: 10, digits: 0}
```
```javascript
   const result = MyFunction("SDV503")
   console.log(result) //{letters: 3, digits: 3}
```
